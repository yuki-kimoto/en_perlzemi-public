    Scalar context

    Perl has a concept called context, and the context that is evaluated as a scalar is called a scalar context .

    Assignment to scalar variable

    Assignment to scalar variable is a scalar context. Array returns the length of the array when evaluated in a scalar context. 

    my $ num = @values;
</ pre>

<h4> Comparison operator term </ h4>

The left and right terms of <a href="/blog/20080318120585.html"> comparison operator </a> are scalar contexts.

<pre>
$ x <@values
$ x == @values
</ pre>

I often see it in the loop of <a href="/blog/20170521149524.html"> for statement </a>.

<pre>
for (my $ i = 0; $ i <@nums; $ i ++) {
  ...
}
</ pre>

<h4> if condition part </ h4>

The conditional part of <a href="/blog/20161023147731.html"> if </a> is the scalar context. In the following, @values ​​is evaluated in scalar context, so it returns the number of arrays.

<pre>
if (@values) {
  ...
}
</ pre>
