<h4> Multiple return values ​​</ h4>

In Perl, using <a href="/blog/20100308126967.html"> list </a> or <a href="/blog/20161012147627.html"> array </a> will return multiple values. I can.

<pre>

<h4> What is the difference between return and exit? </ H4>

While return is used to return to the caller from within a subroutine, exit terminates the program itself. Be sure to use return when you finish the subroutine. It is rare to explicitly describe exit.
