Cut out a record containing a specific time from the log

This is a sample to cut out a record containing a specific time. 

use strict;
use warnings;

print "(1) Cut out a record containing a specific range of time. \ N";
my @records = (
  '1 7:00:00 aaa',
  '2 8:00:00 iii',
  '3 9:00:00 uuu',
  '4 10:00:00 eee',
  '5 11:00:00 ooo',
);

my $ start_time = '08: 00: 00';

# This time is not included
my $ end_time = '11: 00: 00';

for my $ record (@records) {
  
  # Cut out the time
  if ($ record = ~ / (\ d {1,2}): (\ d {2}) :( \ d {2}) /) {
    # You can compare as a character string by matching the format.
    # '7: 00: 00'-> '07: 00: 00'
    my $ time = sprintf ("% 02s:% 02s:% 02s", $ 1, $ 2, $ 3);
    
    # Cut out the records included from the start time to the end time.
    if ($ time ge $ start_time && $ time lt $ end_time) {
      print $ record. "\ n";
    }
  }
}
</ pre>

<h4> Adjust the time format </ h4>
Use the sprintf function to format the time. By adjusting the length of the character string to the same length, it is possible to compare with the character string.

<h4> Cut out the records included from the start time to the end time </ h4>

ge and lt are operators that compare which comes first in the dictionary order of strings. ge is an abbreviation for greater than or equel and lt is an abbreviation for lower than.
