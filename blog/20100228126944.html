<h4> Change the enclosing character </ h4>

In Perl, use the <a href="/blog/20100305126967.html"> double quote operator </a> and the <a href="/blog/20100304126967.html"> quote operator </a> to describe a string. You can change the enclosing character or use the "m" notation to change the enclosing character in regular expressions. Therefore, you can copy and paste the original data without changing it.

<pre>
#Double quote operator
my $ name = qq (aaa "bbb);

#Quart operator
my $ name = q (aaa'bbb);

# Change the enclosing character using m in regular expression pattern matching
if ($ name = ~ m | http: // |) {
  # ...
}

# Change the enclosing character by replacing the regular expression
$ name = ~ s | http: // ||;
</ pre>

<h4> Regular expression escaped by \ Q </ h4>

If you use the regular expression "\ Q" escape, you can copy and paste it as a simple string without being aware of the regular expression characters.

<pre>
my $ re = "abc [] ...";

if ($ name = ~ / \ Q $ re /) {
  # ...
}
</ pre>

use strict;
