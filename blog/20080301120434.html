split function - split string

You can use the split function to split the string by specifying a delimiter. 

my @ret = split(/,/, $person_csv);

In the first argument, specify the delimiter with regular expression. In the second argument, specify the string you want to split. The return value is a array of the split strings.

Use a regular expression

Since the split function receives a regular expression as the first argument, it is easy to split with multiple spaces. 

my @ret = split(/ +/, $line);

Concatenation of strings

The split function splits a string, but the opposite is to use the join function if you want to concatenate the strings. See the following article for the join function.

  • join function - join array with delimiter

Example

This is an example program of the split function.

CSV data

This is an example to convert CSV format data to an array. 

# Convert CSV format data to array
my $string = 'a,b,c';
my @record = split /,/, $string;

Blank delimiter

This is an example to convert to a data array separated by blanks. 

# Convert data separated by blanks to an array
my $string = 'a b c';
my @record = split /\s+/, $string;
