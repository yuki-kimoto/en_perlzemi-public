Giblog 1.0 release - Websites and blog generation tools that can be managed by Git

Giblog 1.0 release. Giblog is a tool for generating websites and blog generation tools that can be managed by Perl seminar-produced Git.

Giblog Official Website - A tool for creating websites and blogs that can be managed by Git

You can easily create static websites that you can manage with Git.

  • Git management for all pages
  • Advanced SEO measures
  • Responsive support
  • Inquiry form
  • Fast rebuild
  • Advanced security
  • Easy to search and replace articles
  • Compatible with Windows Mac Linux
  • Freely customize with Perl

Rebuild 645 pages in 0.78 seconds. Since all pages can be managed with static files, you can freely edit and replace using Perl.

Since everything is a static page, the strongest security that does not create vulnerabilities such as page rewriting.

Compatible with Windows, Mac and Linux.

Functional requests such as adding a Twitter card can be fulfilled using Perl.

Upload it to Github Pages and your page will be ready to go.

Inquiry form is also supported if the environment can use CGI.

Provides advanced SEO protection proven in Perl seminars by default.

Compatible with smartphones by default. Modern website design can also be supported by modifying CSS.

The official Giblog website is of course made on Giblog!
