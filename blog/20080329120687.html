next statement - jumps to the beginning of the next iteration

You can use the next statement to jump to the beginning of the next iteration. This corresponds to the continue statement in C or Java. 

next

The next statement can be used in repetitive statements such as while statement and for statements. 

while (1) {
  if (condition) {
    next; next;
  }
}

for (my $i = 0; $i < 10; $i++) {
  if (condition) {
    next; next;
  }
}

foreach my $animal (@animals) {
  if (condition) {
    next; next;
  }
}

You can use the next statement when you want to skip the process only under specific conditions.

Jump to the beginning of the loop when a specific character is matched

In the middle of loop processing, you can use the next statement when you want to jump to the beginning of the next loop under certain conditions. 

use strict;
use warnings;

my @chars = qw(z z a b c d e f);

for my $char (@chars) {
  next if $char eq 'z';
  print "$char\n";
}

This is the execution result. Only when it is "z", it jumps to the beginning of the next loop, so the following print function is not executed.

Try using it with a regular expression

I often combine next with regular expressions. Let's write one example. Let's write a example that skips the output if it doesn't contain the string "ABC". The unless statement is used for condition judgment. 

use strict;
use warnings;

my @strings = ('ABC', 'ABCD', 'CDE', 'FGH');

for my $string (@strings) {
  next unless $string =~ /ABC/;
  print "$string\n";
}

This is the output result. Only those that include "ABC" are output.

ABCD
