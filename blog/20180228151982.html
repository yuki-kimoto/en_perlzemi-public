    Variable expansion

    Perl has a feature called variable expansion that allows you to use variables in strings. The string must be enclosed in double quotes. 

    my $ str2 = "AAA $ str1 CCC";
</ pre>

Since you can use <a href="/blog/20161107147852.html"> variables </a> directly in <a href="/blog/20161003147558.html"> string </a>, <a href="/blog/20161003147852.html"> variables </a> It's easier than writing using the a href = "/ blog / 20080221120361.html"> string concatenation operator </a>. Let's use it positively.

Let's write a sample of variable expansion.

<pre>
my $ animal ='cat';

my $ message = "I like $ animal";
</ pre>

"$ Animal" is interpolated and "$ message" becomes "I like cat".

<h4> When a character string follows variable expansion </ h4>

When expanding a variable, if a character string follows the variable, that character string will also be recognized as part of the variable name. In such a case, use "{}" after $ to enclose the variable name.

<pre>
my $ animal ='cat';

my $ message = "dog $ {animal} mouse";
</ pre>

It will be correctly expanded as "dog cat mouse".

<h4> Array variable expansion </ h4>

<a href="/blog/20161012147627.html"> Array </a> can also be interpolated. If the array is variable-expanded, the elements are concatenated with blanks.

<pre>
my @animals = ('dog','cat','mouse');

my $ message = "I like @animals";
</ pre>

"$ Message" expands to "I like dog cat mouse".

<h4> Subroutine variable expansion </ h4>

<a href="/blog/20101130129876.html"> Subroutine </a> is not a variable, so it cannot be interpolated. It is smart to assign to a variable and then use the variable expansion function.

<pre>
my $ total = sum (1, 2);
my $ message = "Number is $ total";
</ pre>

<h4> Variable expansion in here document </ h4>

You can also use variable interpolation in <a href="/blog/20080619121405.html"> Here Document </a>. Enclosing the terminating string in double quotes makes it easier to understand the intention of variable expansion.

<pre>
my $ animal ='cat';

my $ message = << "EOS";
I love you.
I like $ animal.
EOS
</ pre>
