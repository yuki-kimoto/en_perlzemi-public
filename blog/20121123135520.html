    Representing complex numbers / Statistical analysis by PDL

    Complex numbers are represented using the i function, which represents the imaginary unit that becomes available when you read PDL :: Complex .

    For example, to represent the complex number 3 + 2i : 

    use PDL;
use PDL :: Complex;

my $ num = 3 + 2 * i;
</ pre>

When this complex number is output, it will be displayed as follows.

<pre>
3 + 2i
</ pre>

<h3> Use the imaginary unit i </ h3>

PDL can handle imaginary numbers. To handle imaginary numbers, load <b> PDL :: PDL :: Complex </ b>.

<pre>
use PDL :: Complex;
</ pre>

Then the <b> i </ b> function that represents the imaginary unit is imported.

<pre>
i
</ pre>

When the imaginary unit is output, it is output as follows.

<pre>
0 + 1i
</ pre>

The imaginary unit i was a number that squared to -1. Let's square it.

<pre>
my $ ret = i * i;
</ pre>

When this is output, it will be displayed as follows.

<pre>
-1 + 0i
</ pre>

<h3> Four arithmetic operations on complex numbers </ h3>

Perform four arithmetic operations on complex numbers using <b> + </ b>, <b>-</ b>, <b> * </ b>, <b> / </ b> as in normal operations. I can.

<pre>
use PDL;
use PDL :: Complex;

my $ num1 = 1 + 2 * i;
my $ num2 = 3 + 4 * i;
</ pre>

<h4> Japanese </ h4>
<pre>
# 4 + 6i
my $ sum = $ num1 + $ num2;
</ pre>

<h4> Difference </ h4>

<pre>
# 2 + 2i
my $ sub = $ num2- $ num1;
</ pre>

<h4> Product </ h4>

<pre>
# -5 + 10i
my $ multi = $ num1 * $ num2;
</ pre>

<h4> Commerce </ h4>

<pre>
# 0.44 + 0.08i
my $ div = $ num1 / $ num2;
</ pre>

<h3> Calculate the nth root </ h3>

Use the <b> Croots </ b> function to calculate the nth root.

<pre>
use PDL;
use PDL :: Complex;

# 5 cube root
my $ nums = Croots (5, 3);
</ pre>

The output of the calculation result is as follows.

<pre>
[1.85398 + 0.496773i -1.35721 + 1.35721i -0.496773 -1.85398i]
</ pre>

<h3> Convert real numbers to complex numbers </ h3>

Use the <b> r2C </ b> function to convert a real number to a complex number.

<pre>
use PDL;
use PDL :: Complex;

my $ num = r2C 5;
</ pre>

The output result is as follows.

<pre>
5 + 0i
</ pre>
