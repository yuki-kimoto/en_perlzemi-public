Installing Perl on macOS

Perl is already installed in the macOS environment.

Check if Perl is installed with the following command. 

perl -v

If the version information is displayed, it's OK.

Let's get started with Perl.

Installing Perl in the user environment

If you want to use the latest Perl etc., you can also install Perl in your environment.

If you want to install the CPAN module, it is recommended to install Perl in your environment.

The new Mac OS X (mojave) system Perl fails to install Perl modules written in C for security reasons.

If you want to install Perl in its own user environment, you can use a tool called perlbrew or plenv.

Since root authority is not required, it is easy to install and does not pollute the OS environment. You can install and use the latest Perl.

If you're running Perl on a production web service, this is the mainstream method as of 2019.

Both perlbrew and plenv are similar tools for installing Perl in your environment. Please select as you like.

Start up the terminal on macOS, and then follow the steps below.

  • perlbrew - Install Perl in your environment
  • plenv - Install Perl in your environment
