    Install msys2 on Windows and use Perl

    Learn how to install msys2 on Windows and use Perl.

    msys2 is a tool that provides a Linux-like shell environment on Windows . You can develop using Perl in a Linux-like environment.

    msys2 screen

    With msys2, you can use Linux commands such as ls, cat, tar, and git, and UTF-8 can be displayed correctly.

    Since there is a POSIX compatibility layer, the development server of Mojolicious will work properly, assuming booting on Linux.

<h4> Install msys2 </ h4>

Double-click the msys2 icon on your desktop.

Press Next to start the installation. It will take about 3 minutes, so let's wait.

When you press Finish, msys2 will start, but let's press the cross on the upper right to end it.

<h3> Perl environment construction 20 minutes </ h3>

Make Perl available on msys2. Let's install Perl and the list of development environments required for Perl together.

pacman is a package manager for msys2.

<pre>
pacman -S perl perl-CPAN msys2-devel make libcrypt-devel
</ pre>

Copy the above and select "Right click"-> "Paste" in msys2 to paste it. Press Enter to execute the above command.

You will be asked if you want to install it, so press "Enter" to proceed. No matter what you ask, just press "Enter".

The following will be installed. This is the minimum environment required for Perl to work properly. Perl is written in <a href="https://clanguage.perlclub.net/"> C language </a> and is a C language compiler <a href="https://linux" for compiling Perl modules. .perlclub.net/blog/20201216103956.html "> gcc </a> is required.

<h4> Run Perl script </ h4>

The user directory of msys2 is in the following location. Place your Perl script under your user directory.

<pre>
C: \ msys64 \ home \ username
</ pre>

Move the current directory.

<pre>
cd C: \ msys64 \ home \ username
</ pre>

Create a file called "test.pl" here. Let's write the following.

<pre>
print "Hello, World! \ N";
</ pre>

Let's run a Perl script.

<pre>
perl test.pl
</ pre>

If you see something like the following, Perl is successfully installed and the program is running.

<pre>
Hello, World!
</ pre>

<h4> Install CPAN module </ h4>

To make sure your Perl environment is built correctly, install Mojolicious running pure Perl and JSON :: XS written in XS.

Some CPAN modules are expected to work in Unix / Linux environments. Make sure the symlink works properly on msys2 with the following settings.

<pre>
export MSYS = winsymlinks
</ pre>

At the first time, you will be asked if you want to set cpan automatically, so press "Enter" to proceed. No matter what you ask, just press "Enter".

<pre>
cpan Mojolicious
cpan JSON :: XS
</ pre>

If you can confirm this installation, the work is finished. Thank you for your hard work.

<h3> Troubleshooting </ h3>

When the msys2 installation and Perl installation procedures fail, virus checkers (Avast, McAfee, Norton, etc.) may falsely detect msys2 as virus software.

Also, depending on the virus checker, it may not be possible to execute with administrator privileges.

If you encounter this issue, temporarily stop the virus checker and right-click "Run as administrator" before launching the msys2 installer.

If you are using Windows 10, it seems that you can install it without any problem by using the default virus software, Windows Defender.
