Perl download and installation

Learn how to download and install Perl. Learn how to install Perl on Windows, macOS, and Linux.

Installing Perl on Windows

Let's install Perl on Windows. I will show you how to use Strawberry Perl, ActivePerl, and msys2.

Install Strawberry Perl

Strawberry Perl is one of the Perl distributions on Windows. Strawberry Perl is easy to install and comes with a C language compiler "MinGW", a CPAN module installer "cpanm", and a C library that is often used for image processing.

Install Strawberry Perl on Windows

Install ActivePerl

ActivePerl is also one of the Perls configured to use Perl on Windows. This is recommended if you want to link with Windows applications such as Excel or create GUI screens.

Install ActivePerl on Windows

Install msys2 on Windows and use Perl

msys2 is a tool that allows you to build a pseudo Linux environment on Windows.

For example, on Windows, interprocess communication features usually don't work, but on msys2 they work.

This will allow modules that rely on interprocess communication, such as Mojolicious development web server, to work fine.

If you want to experience web development on Windows, this is the recommended method.

Install msys2 on Windows and use Perl

Perl is installed by default on Mac, but you can also install the latest Perl in your environment.

If Perl can be installed in the user environment, it will not pollute the system environment, so it is also the recommended method.
