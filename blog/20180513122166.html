    cos function-find cosine

    You can use the cos function to find the cosine. The angle is easy to specify in radians using π (pi function). 

    use Math :: Trig'pi';

cos (pi / 3);
</ pre>

The pi function to get π in radians is explained in <a href="/blog/20080918122175.html"> Math :: Trig </a>.

<h3> Cos function programming sample </ h3>

This is a programming sample using the cos function.

<pre>
use strict;
use warnings;
use Math :: Trig'pi';

# cos (60 °) = 1/2;
print "cos (π / 3) =". Cos (pi / 3). "\ n";
</ pre>
