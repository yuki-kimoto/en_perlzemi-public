    Install Strawberry Perl on Windows

    Strawberry Perl is one of the commercially available Perl distributions created to take advantage of Perl on Windows. Strawberry Perl is easy to install and comes with a C language compiler "MinGW", a CPAN module installer "cpanm", and a C library that is often used for image processing.

    Strawberry Perl for Windows

    Strawbery Perl can be downloaded without membership registration. Older versions of Perl are also available for download.

    MinGW is available at Linux The standard gcc C language compiler. You can also use gmake, which is a port of GNU make to Windows, with Strawberry Perl. With gcc and gmake, many modules written in C / C ++ published on CPAN can be installed on Windows.

    The CPAN module installer cpanm allows you to quickly and easily install CPAN modules with commands. By defining cpanfile, you can define the required modules.

    Download Strawberry Perl

    Assuming Windows 10, we will explain how to download and install it. The procedure is almost the same for older versions of Windows.

    Download and install Strawberry Perl. The latest Perl for 2021 is Perl 5.32.

    Click Strawberry Perl 5.32 64bit MSI installer from the link below to download Strawberry Perl to your desktop.

    • Strawberry Perl Releases

      MSI installer means the default installer for Windows.

      The version number at the end may be a little different, but don't worry.

      In most cases, the 64-bit version of Windows will work, but if you need the 32-bit version, download the 32-bit version of Strawberry Perl.

      Install Strawberry Perl

      Double-click the downloaded Strawberry Perl.

      The installer will start. It may take some time to calculate the free space, but let's just wait.

      The installer will start, so press "Next" to proceed.

      A check box that says "I accept the terms in the License Agreement" will appear on the way. Check it and press "Next".

      Press "Next" to proceed, and finally click "Install" to start installing Strawberry Perl.

      You'll see a dialog here, "Do you want to allow apps from this unknown publisher to make changes to your device?", Click "Yes".

      It will take about 5 minutes, so let's wait.

      When the progress bar is full, the Strawberry Perl installation is complete.

      Press "Finish" to close the dialog. A Readme will be displayed, but you can close it as well.
