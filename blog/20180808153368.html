Install Strawberry Perl on Windows

Strawberry Perl is one of the commercially available Perl distributions on Windows. Strawberry Perl is easy to install and has a C language compiler "MinGW", a CPAN module installer "cpanm", and C libraries which is often used for image processing.

Strawberry Perl for Windows

Strawbery Perl can be downloaded without membership registration. Older versions of Perl are also available for download.

MinGW is a Windows port of the Linux standard C language compiler gcc. You can also use gmake which is a Windows port of GNU make. With gcc and gmake, many modules written in C/C++ on CPAN can be installed into Windows.

The CPAN module installer cpanm allows you to quickly and easily install CPAN modules with commands. By defining cpanfile, you can define the required modules.
