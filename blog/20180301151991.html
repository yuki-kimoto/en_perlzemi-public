    Dereference

    A dereference is the opposite of creating a reference to retrieve an entity from a reference.

    Dereference array reference

    Use "@ {}" to dereference the array reference . 

    my @array = @ {$ array_ref};
</ pre>

Please be aware that a new <a href="/blog/20161012147627.html"> array </a> is not created, but the entity is being retrieved.

If the inside of "@ {}" is a simple <a href="/blog/20161107147852.html"> variable </a>, you can omit "{}".

<pre>
my @array = @ $ array_ref;
</ pre>

This is a sample program that dereferences an array reference.

<pre>
# Array reference
my $ nums_ref = [1, 2, 3];

#Dereference
my @nums = @ $ nums_ref;
</ pre>

<a href="/blog/20161004147558.html"> standard functions </a> that perform Perl array operations must pass an array, not an array reference. In this case, dereference it and pass it to the function.

<pre>
my $ nums = [1, 2, 3];

my $ num = shift @ $ nums;
</ pre>

<h4> Dereference hash reference </ h4>

Use "% {}" to dereference the <a href="/blog/20180208151809.html"> hash reference </a>.

<pre>
my% hash =% {$ hash_ref};
</ pre>

Please be aware that a new <a href="/blog/20161013147635.html"> hash </a> is not created, but the entity is being retrieved.

If the inside of "% {}" is a simple variable, "{}" can be omitted.

<pre>
my% hash =% $ hash_ref;
</ pre>

This is a sample program that dereferences the hash reference.

<pre>
# Hash reference
my $ score_ref = {math => 19, english => 89};

#Dereference
my% score =% $ score_ref;
</ pre>

Standard Perl hashing functions should pass a hash, not a hash reference. In this case, dereference it and pass it to the function.

<pre>
my $ score = {math => 89, english => 45};

my @keys = keys% $ score;
</ pre>

<h4> Dereference scalar variable reference </ h4>

Use "$ {}" to dereference the <a href="/blog/20161107147852.html"> scalar variable </a> reference.

<pre>
my $ scalar = $ {$ scalar_ref};
</ pre>

If the inside of "$ {}" is a simple variable, "{}" can be omitted.

<pre>
my $ scalar = $$ scalar_ref;

</ pre>

This is a sample program that dereferences the scalar reference.

<pre>
# Scalar reference
my $ string1 = "Hello";

my $ string_ref = \ $ string;

#Dereference
my $ string2 = $$ string_ref;
</ pre>
