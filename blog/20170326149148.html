    printf function --formatted string output

    Use the printf function to set the format and print the string. 

    printf "Number% 03d% .3f \ n", 3, 3.14567;
</ pre>

The above output result is as follows. The integer "3" is padded with 0s to the left so that it has 3 digits. The decimal point "3.14567" is rounded to three decimal places of "3.146".

<pre>
Number 003 3.146
</ pre>

"% 03d" and "% .3f" are called format specifiers. See the <a href="/blog/20081001122296.html"> sprintf function </a> description for more information on format specifiers.

<h3> Output to file </ h3>

Like the <a href="/blog/20170404149131.html"> print function </a>, the fprintf function can also be output by specifying a filehandle using indirect object notation.

<pre>
# Output to standard error output
printf STDERR "Number% 03d% .3f \ n", 3, 3.14567;

# Output to file using file handle
printf $ fh "Number% 03d% .3f \ n", 3, 3.14567;
</ pre>

Note that there are no commas behind the filehandle.

For details on how to output, see the explanation of <a href="/blog/20170404149131.html"> print function </a>.

<h3> Export the formatted string to a variable </ h3>

The printf function is a function that outputs, but you can use the <a href="/blog/20081001122296.html"> sprintf function </a> to write a formatted string to a variable.

<pre>
my $ string = sprintf "Number% 03d% .3f \ n", 3, 3.14567;
</ pre>
