Perl Release Information - Introducing feature additions for each version

Here is a summary of Perl release articles. Please help us to understand the changes between Perl versions. If you write it to work with Perl 5.8.7, you can run it in almost any environment.

Perl 5.36

  • Perl 5.36 is released - warnings enabled, subroutine signatures, try-catch-finally statements, Unicode 14 support

Perl 5.28

  • Perl 5.28 released-Unicode 10.0 support, faster string concatenation, ref function, keys function, faster for loop

Perl 5.26

  • The latest version of Perl, Perl 5.26, has been released. Indentable here-documents, Unicode 9.0 support, long key hash performance improvements

Perl 5.24

  • Perl latest version 5.24 released-faster subroutine and math, Unicode 8.0 support, grammar enhancements

Perl 5.22

  • Perl 5.22 has been released-math related enhancements, method performance improvements, regular expression improvements (2015)

Perl 5.20

  • Perl 5.20 released-support for ipv6, new subroutine syntax, new slice notation, Unicode 6.3 support, etc. (2014)

Perl 5.18

  • Perl 5.18 has been released (2013)

Perl 5.16

  • The latest version 5.16.0 of Perl has been released (2012)

Perl 5.14

  • Perl 5.14 release (2011)
