    redo statement-return to the beginning of the iteration

    You can use the redo statement to return to the beginning of the iteration. 

    redo
</ pre>

This is a sample program using redo. You are back at the beginning of the loop you are in.

<pre>
use strict;
use warnings;

my @nums = (1, 2, 3);

while (1) {
  my $ num = pop @nums;
  print "$ num \ n";
  
  if (@nums) {
    redo;
  }
  else {
    last;
  }
}
</ pre>

<h3> redo is a deprecated syntax </ h3>

As I wrote in <a href="/blog/20091120124667.html"> Perl's modern writing method </a>, the redo statement is not recommended.

The reason is that if you have <a href="/blog/20100123126425.html"> while statement </a>, for statement, last statement, next statement, if statement, you can write any repetition and conditional branching, so redo is positive. Because there is no reason to use it as a target. redo is Perl's own syntax, which is unfamiliar and doesn't give you a better view of your source code.
