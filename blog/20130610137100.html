    How to find the code point of a character's Unicode

    If you want to know the Unicode code point of a character, do the following: The utf8 pragma must be enabled and the source code must be saved in UTF-8. 

    use utf8;

my $ str ='Ah';

# Displayed as hexadecimal
printf "% x \ n", ord ($ str);
</ pre>

You can know the code point of the character with the ord function. The printf function specifies the format <b>% x </ b> and displays it in hexadecimal.

The output result is as follows, and you can see that the code point of the character "A" is 3042.

<pre>
3042
</ pre>

<h3> Easily check with Data :: Dumper </ h3>

There is also a simple way to find out using the Data :: Dumper module. This is because Data :: Dumper converts any character that is not in the ASCII range to a code point when outputting the internal string.

<pre>
use utf8;

my $ str ='Ah';

use Data :: Dumper;
print Dumper $ str;
</ pre>

Unicode code points are output in hexadecimal as shown below.

<pre>
$ VAR1 = "\ x {3042}";
</ pre>
